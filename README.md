## 🚀 STEP-BY-STEP DEPLOYMENT GUIDE

### STEP 1 — Create a GitHub Account
1. Go to **https://github.com** → Click **Sign up**
2. Enter email, create password, choose username (e.g. `ortholite-india`)
3. Verify your email address
4. GitHub account is ready ✅

### STEP 2 — Create a New Repository
1. Click **"+"** (top right) → **"New repository"**
2. Repository name: `ortholite-qms`
3. Set visibility: **Public** ✅ (required for free Pages)
4. Check **"Add a README file"**
5. Click **"Create repository"** ✅

### STEP 3 — Upload All App Files
1. In your repo → Click **"Add file"** → **"Upload files"**
2. Extract the ZIP → Open the `qms-app` folder
3. Select ALL contents and drag into GitHub upload area:
   - index.html, app.html, 404.html, .nojekyll
   - css/ folder, js/ folder, assets/ folder
4. Commit message: `Initial QMS deployment`
5. Click **"Commit changes"** ✅

### STEP 4 — Enable GitHub Pages
1. Go to **Settings** tab → **Pages** (left sidebar)
2. Source → Branch: `main`, Folder: `/ (root)`
3. Click **Save**
4. Wait **2–5 minutes**
5. Your URL appears: `https://YOUR-USERNAME.github.io/ortholite-qms/` ✅

### STEP 5 — Open & Share
Visit your URL on any device — desktop, mobile, tablet!
